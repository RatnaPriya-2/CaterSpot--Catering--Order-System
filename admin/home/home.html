Welcome to the CaterSpot admin panel.

From here, you can manage food items, monitor orders, and oversee overall system activity.

As an administrator, you can add new products, update existing menu items, manage stock availability, and control order statuses based on processing and delivery progress.

All customer orders can be reviewed in detail, including items ordered, payment method, and current order status. You can also update order status such as placed, confirmed, delivered.

Use the navigation menu above to manage products, view orders, and maintain smooth system operations.
